## 每集一句话总结

### #1 Introduction & Setup
Claude Code 的正确定位不是“另一个 AI 编辑器”，而是终端里的协作型 coding agent。

### #2 CLAUDE.md Files & /init
`CLAUDE.md` 是项目级说明书，让 Claude 少猜、多按规则做事。

### #3 Context
Claude Code 的效果高度依赖上下文质量，任务越聚焦，上下文越要干净。

### #4 Tools & Permissions
真正需要掌握的不是工具名，而是权限边界与授权策略。

### #5 Planning & Thinking
Planning 管跨度，Thinking 管复杂度，两者不能混用。

### #6 Slash Commands
Slash command 的本质是把高频优质 prompt 和规则沉淀成可复用工作流。

### #7 MCP Servers
MCP 让 Claude 从“只懂本地代码”升级成“能接外部环境”的开发助手。

### #8 Subagents
Subagent 的价值不是多开 AI，而是做职责隔离、上下文隔离、工具隔离。

### #9 Claude Code with GitHub
Claude Code 接入 GitHub 后，才能真正进入 issue、PR、review 的正式协作流。

### #10 Final Thoughts & Tips
最重要的高级技巧不是快捷键，而是：人始终在环、上下文始终干净、AI 始终只是协作者。

## 最核心的一句话
